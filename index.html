తెలుగు అమ్మ

A simplified 28-letter script for Digital Telugu

About Telugu-Amma

తెలుగు అమ్మ అనేది తెలుగు భాషను డిజిటల్ యుగంలో సులభంగా వాడటానికి సృష్టించిన కొత్త పద్ధతి. మన సంప్రదాయ తెలుగు అక్షరాలు 56 కంటే ఎక్కువ ఉన్నాయి, అందులో గుణింతాలు, ఒత్తులు వంటివి typing లో కష్టం. కానీ ఈ కొత్త Telugu-Ammaలో కేవలం 28 అక్షరాలతో ఏ పదమైనా రాయవచ్చు.

Telugu-Amma is a simplified script inspired from traditional Telugu letters. Instead of complex gunintalu and ottulu, it uses only 28 base letters. This makes it easy to type on phones, WhatsApp, YouTube comments, Instagram, and all digital platforms.

Traditional vs Telugu-Amma

Traditional Telugu uses: అ, ఆ, ఇ, ఈ, ఉ, ఊ, ఎ, ఏ, ఐ, ఒ, ఓ, ఔ… + 200+ forms with gunintalu and ottulu.

Telugu-Amma uses only 28 letters: అ, ఆ, ఇ, ఈ, ఉ, ఊ, ఎ, ఏ, ఐ, ఓ, క, గ, చ, జ, ట, డ, త, ద, న, ప, బ, మ, య, ర, ల, వ, శ, స, హ

Examples:

  • చరణ్ → చఅరఅన
  • రామ → రఆమఅ
  • ప్రాణం → పరఆనమ
  • అమ్మ → అమమ
  • స్నేహం → సనఏహమ

Our Aim

మన తెలుగు భాష WhatsApp, YouTube, Instagram, Facebook, social media వంటి ప్రతి డిజిటల్ ప్లాట్‌ఫార్మ్‌లో వాడబడాలి. ఇంగ్లీష్ మాత్రమే వాడటం వల్ల తెలుగు కనుమరుగవకుండా, Telugu-Amma డిజిటల్ యుగానికి సరిపోయే సులభమైన రూపం ఇస్తుంది.

The same method can be applied to Tamil, Kannada, Malayalam, and Hindi to make all mother languages stronger in digital life.
